# MovieLens-Analysis
Pandas, MySQL and Power BI Analysis on MovieLens Dataset.

# MovieLens Dataset Analysis

This project contains:
- SQL Analysis (Top-rated movies, genres, active users, etc.)
- Pandas Analysis (genre trends, pivot tables, rating analysis, variance detection)
- Power BI Dashboard (User Insights, Genre Trends, KPI Cards)

## Folder Structure
- SQL_Tasks/ → Contains all SQL queries
- Pandas_Tasks/ → Jupyter notebook & python code
- PowerBI_Dashboard/ → Power BI report

Dataset used: MovieLens Latest Small Dataset.
